# 📖 About the Project

The **Cloud Data Redundancy Removal System** is a modern full-stack web application developed to eliminate duplicate records before they are stored in the database. The system helps maintain clean, accurate, and efficient data by automatically detecting duplicate email entries and preventing redundant information from being inserted.

Built using **FastAPI**, **SQLite**, **HTML**, **CSS**, and **JavaScript**, this project demonstrates real-world implementation of CRUD operations, REST APIs, client-server communication, and database management.

This application is suitable for organizations that require reliable data validation and efficient record management while minimizing storage redundancy.

# ⚙ Installation Guide

## Step 1

Clone the repository

```bash
git clone https://github.com/mohammadzunaid333-hash/CodeAlpha-Data-Redundancy-Removal-System.git
```

---

## Step 2

Move into the project folder

```bash
cd CodeAlpha-Data-Redundancy-Removal-System
```

---

## Step 3

Create a Virtual Environment

```bash
python -m venv venv
```

---

## Step 4

Activate the Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

---

## Step 5

Install Required Packages

```bash
pip install -r requirements.txt
```

---

## Step 6

Start the FastAPI Server

```bash
uvicorn app:app --reload
```

---

## Step 7

Open your browser

```
http://127.0.0.1:8000
```
